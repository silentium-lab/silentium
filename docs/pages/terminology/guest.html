Гость

Гость - это посетитель, который является функцией или объектом без introduction или объектом с методом introduction, который возвращает значение 'guest'

Гость получает значение от источника данных один раз. Но библиотека имеет механики на случай если у источника данных нет данных на текущий момент, то можно задать гостю значение при первом появлении данных. Для этих механик смотрите: GuestPool, PatronOnce, SourceEmpty
