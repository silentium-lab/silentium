Silentium = молчание

С латинского языка silentium переводится как молчание. Основная особенность библиотеки silentium заключается в том, что объект, если у него нет хорошего ответа на вопрос может не давать плохой ответ, например - null. Такой подход позволяет объектам лучше контроллировать свою часть работы, а тем объектам которые полагаются на работу других объектов позволяет доверять полученным значениям, не размазывая множество условий проверки значений по всему приложению.

Основа реализации в том, что объекты разделяются на две категории: Источники и Посетители.
Источники - это объекты с методом .value или функции ожидающие в первом аргументе посетителя.
Посетители - это объекты с методом .give или функции ожидающие в первом аргументе значение.

Начало работы

Чтобы установить библиотеку воспользуйтесь пакетным менеджером npm или другим более предпочтительным для вашего приложения. 

$ npm install silentium
