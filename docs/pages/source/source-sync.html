Класс SourceSync

SourceSync на GitHub

Класс SourceSync позволяет построить композицию классов и синхронно прочитать полученное значение. Под капотом класс основан на GuestSync. Класс будет полезен в логике тестов, чтобы проверять композиции сразу синхронно в тесте.

Если сравнивать пример из документации GuestSync c примером ниже, то видно что SourceSync более лаконичный.

Пример 1.

import { sourceChangeable, SourceSync } from "silentium";

const syncSource = new SourceSync(sourceChangeable("Успешно установлено синхронно!"));
document.querySelector(".guest-result").textContent = syncSource.syncValue();

Результат 1.

Значение установится синхронно!
