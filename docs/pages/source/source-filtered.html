Класс SourceFiltered

SourceFiltered на GitHub

Если в программе нужно отдавать значения по-условию, то можно использовать класс SourceFiltered, который возвращает значения только если функция-предикат возвращает true

Пример 1.

import { sourceChangeable, sourceFiltered, patron } from "silentium";

const guest = v => {
  const r = document.querySelector(".guest-result");
  if (r) {
    r.textContent = `auth = ${v}`;
  }
}

const s = sourceChangeable(false);
const authComplete = sourceFiltered(s, (v) => v === true);
authComplete.value(patron(guest));

setTimeout(() => {
  s.give(true);
}, 3000);
