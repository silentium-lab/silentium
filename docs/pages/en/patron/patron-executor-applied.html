PatronExecutorApplied

PatronExecutorApplied on GitHub

The PatronExecutorApplied under the hood uses gutexecutorapplied, but it is a patron.

Is useful when we work with data sources with a pool of cartridges that can send new values.

Example 1.

import { guest, PatronExecutorApplied, patron, sourceChangeable } from "silentium";

window.source = sourceChangeable(20);

const loadingSource = sourceChangeable('');

const guest = guest((value) => {
  document.querySelector(
    ".guest-result"
  ).textContent = `value is ${value}`;
});

loadingSource.value(patron(guest));

window.source.value(
  new PatronExecutorApplied(guest, (guestFn) => {
    return v => {
      loadingSource.give('Loading...');
      setTimeout(() => {
        guestFn(v);
      }, 5000)
    }
  })
);
