PatronPool class

PatronPool on GitHub

Designed to store a list cartridges to pass values to

Useful for organizing a list of subscribers for updating data of some kind source . The PatronPool class itself is quite low-level and desirable. use the Source class, which is exactly the source of the values ( Source has a PatronPool running under the hood)

Example 1.

import {
    PatronPool,
    patron
} from 'silentium';

const pool = new PatronPool();

pool.add(patron((value) => {
    document.querySelector('.patron-pool-result1').textContent = value + ' 1';
}));
pool.add(patron((value) => {
    document.querySelector('.patron-pool-result2').textContent = value + ' 2';
}));

pool.give('Value from the pool');

Result 1.
