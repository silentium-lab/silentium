PatronApplied class

PatronApplied on GitHub

The PatronApplied class under the hood uses GuestApplied, only it is a patron.

Is useful when we work with data sources with a pool of cartridges that can send new values.

Example 1.

import { guest, PatronApplied, sourceChangeable } from "silentium";

window.source = sourceChangeable(20);

const g = guest((value) => {
  document.querySelector(
    ".guest-result"
  ).textContent = `Double value is ${value}`;
});

window.source.value(
  new PatronApplied(g, (x) => x * 2)
);
