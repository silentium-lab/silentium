Sometimes it is necessary to connect different objects data source and user in one object as a common data source that can be changed

For example, the data source of structure X may be object A, and a visitor receiving a new version of structure X may be object B. In this case, you can combine objects A and B into one a common SourceChangeableType, which makes it easier to manipulate the structure X

For such cases, the sourceDynamic is suitable, combining GuestType and SourceType
