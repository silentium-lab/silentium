SourceFiltered

SourceFiltered on GitHub

If the program needs to return values conditionally, then you can use the SourceFiltered class, which returns values only if the predicate function returns true

Example 1.

import { sourceChangeable, sourceFiltered, patron } from "silentium";

const guest = v => {
  const r = document.querySelector(".guest-result");
  if (r) {
    r.textContent = `auth = ${v}`;
  }
}

const s = sourceChangeable(false);
const authComplete = sourceFiltered(s, (v) => v === true);
authComplete.value(patron(guest));

setTimeout(() => {
  s.give(true);
}, 3000);
