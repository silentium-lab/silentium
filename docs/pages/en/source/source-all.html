SourceAll class

SourceAll on GitHub

Designed to combine results from multiple data sources into a single source

Useful behavior in cases where there are many data sources on the basis of which you need to make a decision at the moment when all sources give their data

Example 1.

import {
    source,
    sourceAll
} from 'silentium';

const all = sourceAll({
  one: source(1),
  two: source(2),
});

all.value(({one, two}) => {
    document
        .querySelector('.guest-result')
        .textContent = `Result = ${one + two}`;
});

It is also possible to collect the results in an array rather than an object.

Example 1.

import {
  SourceChangeable,
  sourceAll,
  source
} from 'silentium';

const all = sourceAll([
  source(1),
  source(2),
]);

all.valueArray((numbers) => {
    document.querySelector('.guest-result-array').textContent = `Result = ${numbers}`;
