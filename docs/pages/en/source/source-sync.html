SourceSync

SourceSync on GitHub

The SourceSync class allows you to build a composition of classes and synchronously read the received value. Under the hood, the class is based on GuestSync. The class will be useful in the logic of tests to check compositions immediately synchronously in the test.

If we compare the example from the GuestSync documentation with the example below, we can see that SourceSync is more concise.

Example 1.

import { sourceChangeable, sourceSync } from "silentium";

const syncSource = SourceSync(sourceChangeable("Successfully installed synchronously!"));
document.querySelector(".guest-result").textContent = syncSource.syncValue();

Result 1.

The value will be set synchronously!/span>
