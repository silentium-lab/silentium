SourceOnce

SourceOnce on GitHub

The SourceOnce class is used in cases where you need to create a data source in which there will be no value at the start, it will be installed later but only once, subsequent installations new values will be ignored.

Example 1.

import { sourceOnce, patron } from "silentium";

const source = sourceOnce();

source.value(patron((v) => {
    document.querySelector(".guest-result").textContent = v;
}));

setTimeout(() => {
    source.give(10)
}, 1000);

// The code below has
no setTimeout(() effect => {
    source.give(20)
}, 2000);

setTimeout(() => {
    source.give(30)
}, 3000);
