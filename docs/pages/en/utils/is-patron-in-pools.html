isPatronInPools function

isPatronInPools on GitHub

This function is very useful in order to check whether the cartridge is in some kind of bullet. In order to debug possible problems or acceptance scenes some solutions can use this feature.

For example, if a guest comes to the object method and we give this guest a value from the Source, then if new values appear in the source, we can check whether the cartridge exists in the pool, this happens when using SourceAll in combination with GuestCast, because in this case the source It will communicate not with the original patron, but with an intermediate one.
