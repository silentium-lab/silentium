Sometimes we want to get an addiction that we can lead with. private communication, that's exactly what the type was created for. href="https://github.com/silentium-lab/silentium/blob/main/src/Private/Private.ts#L4-L6" > PrivateType

By creating a dependency on PrivateType, we say that we plan to send private messages that no one else should read. To achieve this goal of sending private messages, there are classes Private and PrivateClass. Private creates an object from a function, and PrivateClass creates a private object based on the class.

Example 1.

class SomeAction {
  public constructor(private source: PrivateType<SourceType<number>>) { }

  do() {
// Here we are sure that the 'hello' was transmitted personally and no one will receive it
    this.source.get('hello').receiving(theNum => {
      console.log('do something with num', theNum);
    })
  }
}
