GuestDisposable class

GuestDisposable on GitHub

Designed for dynamic creation guest with the functionality of removing it from ammo pool , in case the guest becomes patron

Useful in cases where a guest has done his job and needs to be removed from the pool cartridges, but only if the guest has become a patron. The Patron class has a method disposed, which is taken from the guest. In the PatronPool logic, after each transfer of a value to a patron, it is checked whether it is necessary to remove this patron from the pool by calling the disposed method

Instead of the GuestDisposable class, you can use the function. href="/utils/remove-patron-from-pools" > removePatronFromPools - in order to remove a cartridge from all pools. or the function href="/utils/is-patron-in-pools" > isPatronInPools - checks if there is a cartridge in any bullet

Example 1.

import { sourceChangeable, patron, guestDisposable } from "silentium";

  const source = sourceChangeable("One");

  source.value(
    patron(
      guestDisposable(
        (value) => {
          document.querySelector(".guest-result").textContent = value;
        },
        (value) => {
          return value !== "One";
        }
      )
    )
  );

  setTimeout(() => {
    source.give("Two");
  });

  setTimeout(() => {
    source.give("Three");
  }, 100);
