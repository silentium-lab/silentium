GuestExecutorApplied

GuestExecutorApplied on GitHub

Sometimes we need to control not the value that comes to the guest, but the process of transferring the value, for example, if we want to apply the debounce function or the throttle function to the guest in order to reduce the number of triggers of the guest logic.

Example 1.

import { guest, guestExecutorApplied, sourceOf } from "silentium";

const guest = guest((value) => {
  document.querySelector(
    ".guest-result"
  ).textContent = `value is ${value}`;
});

sourceOf(40).value(
  guestExecutorApplied(guest, (guestFn) => {
    return v => {
      setTimeout(() => {
        guestFn(v);
      }, 5000)
    }
  })
);
