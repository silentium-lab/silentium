Guest class

Guest on GitHub

Designed to create new instances GuestObjectType from the callback function

This behavior is useful in cases where you need to dynamically build user based on the function, using some variables from the scope where it was created function

Example 1.

import { guest } from 'silentium';

class Timestamp {
    seconds(guest) {
        guest.give(Date.now())
    }
}

const timestamp = new Timestamp();
timestamp.seconds(guest((timestamp) => {
    console.log('timestamp is', timestamp);
}));

The ability to create a visitor object from the function was the only one in the first versions of the library. Currently, you can also pass the callback function as a visitor, since the function corresponds to the type GuestType This allows you not to create unnecessary object wrappers.

Example 2.

import { Guest } from 'silentium';

class Timestamp {
    seconds(guest) {
        give(Date.now(), guest);
    }
}

const timestamp = new Timestamp();
timestamp.seconds((timestamp) => {
    console.log('timestamp is', timestamp);
});
