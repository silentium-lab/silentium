Utilities

This section contains documentation on auxiliary functions and classes Patron libraries

The main library classes are listed in the top menu of the website (this documentation), and all auxiliary modules will be described here

It should be noted that the main classes of the library are: Guest, Patron, Source and their derivatives work at the level of human logic, and the classes from this section (utils) are designed to to work at the computer level.

In the left menu of this section you will find all utilities and their descriptions
